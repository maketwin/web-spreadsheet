## 特性

- **4 层架构** — Facade / Commands / Store / Renderer 分层清晰
- **Canvas 渲染** — VirtualScroller + DirtyRegionTracker 高性能渲染
- **公式引擎** — 32+ 内置函数，依赖图增量重算
- **Command 模式** — 完整 Undo/Redo 支持
- **插件系统** — 可扩展的 Plugin API
- **暗色模式** — CSS 变量驱动的主题切换
- **xlsx 导入导出** — 基于 SheetJS 的真实 Excel I/O
- **IndexedDB 自动保存** — 1.5s 防抖自动持久化
